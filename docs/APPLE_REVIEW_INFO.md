## Review notes

XpressIntra is an internal employee app for XpressBudet staff. It is used for employee communication, operational notices, voluntary location sharing, pickup tasks, vehicle information, personal workday overview and internal information guides.

The app requires login. Please use the provided test account during review. Location sharing is voluntary and only starts when the user actively enables it or starts a task where location is relevant. Camera/photo access is only used when the user actively selects or takes a photo for profile, chat, posts, logbook or tasks.
