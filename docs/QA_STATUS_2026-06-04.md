# XpressIntra QA-status 2026-06-04

## Samlet vurdering

Appkoden er i god stand til intern test og ny medarbejderpilot, men live Supabase mangler stadig nogle nyere tabeller til dokumenter, GDPR-accept, logbog og feedback. Kerneflowet med login, profil, chat, GPS, media-storage, standardkode-onboarding, release-check og web/Android/iOS-sync tester grønt.

- Browserens visuelle automatik kunne ikke køre i Codex-sandkassen, så rigtig kliktest på mobil skal stadig udføres manuelt.

## Go-live anbefaling

Før bred intern brug bør den fulde Supabase SQL-pakke køres eller repareres, så de manglende tabeller også findes live. Derefter bør appen testes med mindst to rigtige medarbejderprofiler: én lastbil og én varebil. GPS, chat, profilbillede, standardkode-login og første adgangskodeskift bør testes på rigtige telefoner.
