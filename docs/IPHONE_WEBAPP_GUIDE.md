# iPhone-guide til XpressIntra

XpressIntra kan bruges på iPhone uden App Store, når medarbejderen åbner webappen i Safari og lægger den på hjemmeskærmen.

## Sådan gør medarbejderen

1. Åbn Safari på iPhone.
2. Gå til `https://xpresshub-seven.vercel.app`.
3. Tryk på Del-knappen nederst i Safari.
4. Vælg `Føj til hjemmeskærm`.
5. Tryk `Tilføj`.
6. Åbn IntraBudet/XpressIntra fra hjemmeskærmen og log ind.

## Vigtigt

- Brug Safari på iPhone. Andre browsere kan vise siden, men Safari er den rigtige vej til hjemmeskærm-installation.
- iPhone-versionen bruger samme login, Supabase-data, chat, GPS, opslag og Information-panel som pc/web og Android.
- Hvis GPS skal bruges, skal medarbejderen give Safari/XpressIntra tilladelse til lokation.
- Hvis billeder skal bruges, skal medarbejderen give adgang, når appen spørger.
- App Store/TestFlight er ikke nødvendig for intern brug via Safari.

## Når appen opdateres

Når webappen udgives igen, får iPhone-brugere den nye version via samme link/hjemmeskærms-app. Hvis en iPhone hænger i gammel cache, kan medarbejderen åbne Safari, gå til app-linket og genindlæse siden.
